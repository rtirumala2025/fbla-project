# Demo Video Reference

- **Purpose:** Fail-safe playback if live demo experiences connectivity or hardware issues.
- **Suggested Length:** 6–7 minutes following the script in `docs/presentation.md`.
- **Required Chapters:**
  1. Registration and theme toggle
  2. Pet care loop with notifications
  3. Mini-game reward overlay
  4. Shop purchase + sound feedback
  5. Analytics dashboard and CSV export

- **Recording Tips:**
  - Capture cursor and keypress hints.
  - Keep resolution at 1080p (1920×1080) with 60 fps if possible.
  - Run with ambient audio muted to avoid venue interference.
  - Include captions or narration describing the learning outcomes.

- **Storage:**
  - Upload to an accessible cloud folder (e.g., Google Drive, OneDrive).
  - Set permissions to “Anyone with the link can view”.
  - Paste the final share link below:

```
Video Link: <insert once uploaded>
Backup USB Filename: virtual-pet-demo.mp4
```
